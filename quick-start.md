# ⚡ Quick Start Guide

A fast, simple setup for your watch.

---

## 1. Set the Time
1. Go to Timekeeping mode  
2. Hold **D** for 2 seconds  
3. Use **D** to move through settings  
4. Use **C** to change values  
5. Press **A** to save  

---

## 2. Set the Date
1. Go to Date & Temperature mode  
2. Hold **D** for 2 seconds  
3. Use **D** to switch Month → Day → Year  
4. Use **C** to change values  
5. Press **A** to save  

---

## 3. Turn Alarm On/Off
- Go to Alarm mode  
- Press **C** to toggle alarm  
- Hold **C** to test alarm  

---

## 4. Use the Stopwatch
- **D** → Start / Pause  
- **C** → Split / Reset  
- **A** → Exit  

---

## 5. Backlight
Press **B** anytime.

---

## 6. Hourly Chime
Hold **C** in Timekeeping mode to toggle.
